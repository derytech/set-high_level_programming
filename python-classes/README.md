# Python - Classes and Objects

This repository contains Python projects focused on Object-Oriented Programming (OOP).

The tasks cover the creation and use of classes, objects, attributes, methods, and other concepts related to Python classes.

## Learning Objectives

By completing these tasks, I will learn:

- What is Object-Oriented Programming
- How to define a class in Python
- How to create objects and instances
- How to work with attributes and methods
- How to use Python classes effectively

## Requirements

- Python 3.8.5
- Ubuntu 20.04 LTS
- pycodestyle 2.8.*

## Files

- `0-square.py` - Defines an empty class `Square`.
