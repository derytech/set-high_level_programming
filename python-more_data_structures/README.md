# Python - More Data Structures: Set, Dictionary

This project covers advanced Python data structures, including:

- Sets
- Dictionaries
- Lambda functions
- map()
- filter()
- reduce()

The tasks focus on manipulating collections efficiently using Pythonic techniques such as list comprehensions and dictionary methods.

## Learning Objectives

- Understand and use sets and dictionaries.
- Perform operations on dictionaries.
- Use lambda functions with map(), filter(), and reduce().
- Write clean, readable, and efficient Python code.
